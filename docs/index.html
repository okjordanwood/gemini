System Requirements

Minimum

OS: Windows 7 or later / macOS Mojave or later / Ubuntu 18.04+
Processor: Dual-core 1.6 GHz
Memory: 2 GB RAM
GPU: Integrated graphics (e.g., Intel HD Graphics 2000 or equivalent)
Storage: 500 MB available space
Internet: Broadband connection

Recommended

OS: Windows 10 / macOS Catalina+ / Recent Linux distro
Processor: Dual-core 2.5 GHz
Memory: 4 GB RAM
Storage: 1 GB available space
Internet: High-speed connection
