About the Project

This project was developed as part of Capstone Project at University of South Carolina. The goal was to create an application that is a platformer game where the room's puzzle are not so much of moving around the area, but having the area being affected around you.

We started development in 9/2024 and released the first version in 4/2025. Since then, we've been continuously improving the app based on user feedback and adding new features.

The source code for this project is available on GitHub. Feel free to check it out and report issues if you're interested!
